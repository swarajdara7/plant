# 🌱 PlantPal – Smart Plant Care Management System

## 📌 Overview

PlantPal is a full-stack web application developed to help users identify plants and manage plant care efficiently. The system provides automated reminders for watering and maintenance schedules, helping users maintain healthy plants through a simple and responsive interface.

---

## 🚀 Features

* 🌿 Plant identification and management
* ⏰ Automated watering and maintenance reminders
* 📱 Responsive and user-friendly interface
* 🔄 CRUD operations for plant records
* 🗄️ Secure database connectivity and management

---
